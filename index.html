Quick Start Guide

Setting Up Your Business Profile

  1. Tap the Settings tab at the bottom of the screen
  2. Enter your business name, contact information, and upload your logo
  3. Set your payment processing fee percentage if applicable
  4. Upload your Venmo QR code for easy payments from customers

Adding Your Artists

  1. Navigate to the Artists tab at the bottom of the screen
  2. Tap the 'Add Artist' button
  3. Enter artist details including name, type, hourly rate, and per-attendee rates
  4. Add custom artist types if the default options don't fit your needs

Managing Artist Availability

  1. Navigate to an artist's detail page
  2. Tap 'Manage Schedule' to view their availability
  3. Add new schedule entries to mark when they're unavailable
  4. Set up recurring unavailability patterns for regular commitments
  5. The system automatically checks for scheduling conflicts when booking artists for events
  6. Multi-day events will automatically block artists across all event days

Managing Customers

  1. Navigate to the Customers tab
  2. Add new customers with the + button
  3. Flag problematic customers with 'Do Not Book' option
  4. Set up custom invoice patterns using # characters for sequential numbers (e.g., 'SMITH-####')
  5. Search and view customer history by tapping on a customer

Booking Events

  1. Tap the Events tab and then the + button
  2. Fill in event details and select an existing customer or add a new one
  3. Toggle 'Multi-day Event' to ON for events spanning multiple days
  4. Choose artists, set rates, and select your billing type (hourly or per-attendee)
  5. Set up event details including setup time (which isn't billed)
  6. Apply discounts if needed (by hours or flat amount)
  7. The system will automatically check for artist scheduling conflicts

Creating Invoices

  1. From an event detail page, tap 'Create Invoice'
  2. Review line items and make any adjustments
  3. Add processing fees if necessary
  4. Use custom invoice numbers or let the system generate sequential ones
  5. For customers with custom invoice patterns, the system will automatically suggest the next number in their sequence
  6. Email the invoice directly to your customer

Recording Customer Payments

  1. From an invoice, tap 'Record Payment'
  2. Enter payment details and method
  3. Record partial payments if the customer hasn't paid in full
  4. Generate a receipt for the customer if needed

Managing Artist Payments

  1. Navigate to the Artist Payments tab at the bottom of the screen
  2. To record a new payment, tap the 'Add' button
  3. Select the event and artist, and the app will suggest a payment amount
  4. Record payment details including date, method, and status
  5. View payment history for specific artists from their profile page

Managing Gift Certificates

  1. Navigate to the Gift Certificates tab
  2. Create new certificates with the + button
  3. Set amount, recipient details, and optional expiration date
  4. Print or email the certificate to the purchaser or recipient

User Guide

Dashboard

Overview of your upcoming events and business activities

Using the Dashboard

  1. Open the app and the Dashboard appears by default
  2. View your upcoming events in the calendar widget
  3. Check quick stats for earnings, upcoming events, and payments due
  4. Use quick action buttons to create new events, invoices, or payments

Customizing Your Dashboard

  1. Tap and hold on any widget to enter edit mode
  2. Drag widgets to rearrange their order
  3. Tap the gear icon on any widget to customize its settings
  4. Pull down to refresh and update all dashboard data
  5. Tap the date range at the top to change the time period for statistics

Using Quick Actions

  1. Tap the '+' button at the bottom of the Dashboard
  2. Select from available quick actions: New Event, New Invoice, Record Payment, etc.
  3. Fill in the required information in the form that appears
  4. Tap 'Save' to create the new item and return to the Dashboard
  5. Look for the confirmation indicator showing the action was completed

Events

Manage your calendar and client bookings

Creating a New Event

  1. Tap the 'Events' tab at the bottom of the screen
  2. Tap the '+' button in the top-right corner
  3. Fill out the event details form with customer info, location, date, and time
  4. Select billing type (hourly or per-attendee)
  5. Add setup time (which won't be billed)
  6. Tap 'Select Artists' to add painters to the event
  7. Apply any discounts if needed
  8. Tap 'Save Event' to complete

Managing Event Status

  1. Open the event details by tapping on it in the Events list
  2. Tap 'Update Status' button
  3. Select new status: Inquiry, Scheduled, Confirmed, Completed, or Cancelled
  4. Add optional notes about the status change
  5. Tap 'Update' to save the changes

Creating Multi-Day Events

  1. In the Event creation screen, enter the starting date and time
  2. Toggle 'Multi-day Event' to ON
  3. The 'End Time' field will change to show both date and time options
  4. Select the ending date and time for the event
  5. When you assign artists, they will be blocked for the entire date range
  6. Multi-day events appear on all applicable days in the calendar

Artists

Manage your team of artistic talent

Adding a New Artist

  1. Tap the 'Artists' tab at the bottom of the screen
  2. Tap the 'Add Artist' button at the bottom
  3. Enter the artist's name, contact information, and select their type
  4. To create a custom artist type, tap the 'New Type' button next to the type selector
  5. Set the hourly rate and per-attendee rate (if different)
  6. Add any notes about the artist
  7. Tap 'Save Artist' to complete

Managing Artist Availability

  1. Navigate to the artist's profile by tapping on them in the Artists list
  2. Tap 'Manage Schedule' under the Availability section
  3. Tap 'Add Schedule Entry' button
  4. Set the date range and select 'Unavailable' status
  5. For recurring unavailability, toggle 'Repeating Schedule' and choose pattern
  6. Select repeat frequency (Daily, Weekly, Monthly, or Custom)
  7. For weekly repeats, select specific days of the week
  8. For monthly repeats, choose specific dates or patterns
  9. Set an end date for the recurring pattern or select 'No End Date'
  10. Add notes if needed (e.g., 'Teaching class')
  11. Tap 'Save' to add to their schedule

Artist Payments

Manage payments to your artists

Recording an Artist Payment

  1. Tap the 'Artist Payments' tab at the bottom of the screen
  2. Tap the 'Add' button at the bottom of the screen
  3. Select the event from the list (only completed events are shown)
  4. Select the artist who worked at the event
  5. Review the suggested payment amount (calculated automatically)
  6. Adjust the amount if needed or tap 'Use Suggested Amount'
  7. Choose payment method (cash, check, Venmo, etc.)
  8. Select payment status (paid or pending)
  9. Add optional notes
  10. Tap 'Record Artist Payment' to save

Understanding Suggested Payment Amounts

  1. When recording a new artist payment, select the event and artist
  2. The app automatically calculates and displays a suggested payment amount
  3. For hourly billed events, this is (artist hourly rate × billable hours)
  4. For per-attendee events, this is (artist per-attendee rate × number of attendees)
  5. The suggestion is based on rates set in the artist's profile
  6. Event-specific artist rates will override the default rates for suggestions

Frequently Asked Questions

General Questions

How do I get started with the app?

Start by setting up your business profile in the Settings tab. Then add your artists, create customers, and begin booking events. The Quick Start Guide provides step-by-step instructions for getting up and running quickly.

Can I use the app on multiple devices?

Yes, you can use the app on all your iOS devices by signing in with the same Apple ID. Your data will sync automatically between devices using iCloud.

Is my data backed up?

Yes, your data is automatically backed up to iCloud if you have iCloud enabled on your device. You can also manually export a backup file from Settings > Backup & Restore.

Billing and Payments

How do I create an invoice for an event?

Open the event details and tap 'Create Invoice'. The app will automatically generate line items based on the event details. You can then review and modify before finalizing.

Can I create custom invoice numbers?

Yes, when creating an invoice, toggle 'Use Custom Invoice Number'. You can then enter your preferred format, or use the suggested customer-specific format. For customers with a custom invoice pattern (like 'ABC-####'), the system will automatically generate the next sequential number.

How do I set up a custom invoice pattern for a customer?

Open the customer's profile, tap 'Edit Customer', and scroll to the 'Custom Invoice Numbering' section. Enter a pattern using # as placeholders for sequential numbers (e.g., 'SMITH-####'). The system will replace the # characters with incrementing numbers for each new invoice.

Artist Management

Can I track when artists are unavailable?

Yes, on the artist's profile page, tap 'Manage Schedule' and add schedule entries with 'Unavailable' status. You can set one-time blocks or recurring unavailability patterns (daily, weekly, monthly). For weekly patterns, you can select specific days of the week, and for monthly patterns, you can choose specific dates or patterns like 'first Monday'.

What happens if I try to book an unavailable artist?

The app will warn you about the conflict when you try to add an artist to an event during a time they're marked as unavailable. You can choose to override this warning if needed.

Events

How do I create a multi-day event?

When creating a new event, toggle 'Multi-day Event' to ON. This will allow you to select both a start date/time and an end date/time that can span multiple days. The system will handle artist scheduling across all days of the event.

What does 'setup time' mean for an event?

Setup time is the period before the actual event starts when artists arrive to prepare. This time is tracked but not billed to the customer automatically. The billable duration is calculated from the event start time to end time.
